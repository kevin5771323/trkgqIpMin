# 项目简介

## 核心功能
- 基于YOLOv8的电子元器件检测系统，可识别['Capacitor', 'Inductor', 'Led', 'Resistor', 'Transistor']五类电子元器件
- 支持多种输入源：单张图片、图片文件夹、视频文件及本地摄像头
- 提供用户友好的界面，实时显示目标位置、目标总数和置信度等信息
- 支持批量检测功能，可查看所有检测结果
- 可自定义检测参数：置信度和IOU阈值
- 提供模型选择功能，支持显示标签和原图切换
- 支持检测结果保存功能

## 适用场景
- 电子元器件质量检测与分类
- 生产线自动化检测
- 库存管理与元器件识别
- 教学研究与实验演示
- 开发人员学习目标检测应用

## 技术特点
- 使用Pycharm/VSCode+Anaconda开发环境
- 基于Python 3.9，依赖opencv-python、PyQt5、ultralytics和torch1.9等库
- 提供完整的源代码、UI界面源文件、图标及数据集
- 包含2426张标注数据集，支持直接运行和模型训练
- 配置成功环境可直接使用，无需额外开发
